## Boolean Comparisons

| Comparison |        Description        |
| :--------: | :-----------------------: |
|    `==`    |         equal to          |
|    `!=`    |       not equal to        |
|  `equals`  |   equal to (for String)   |
| `!equals`  | not equal to (for String) |

- Never use `==` or `!=` to compare String values (get weird results).
    - Use `equals` or `!equals` instead.
